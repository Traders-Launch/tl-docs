# Payout Processing SOP

## Overview

How to process trader payouts weekly.

## Prerequisites

* Access to admin dashboard
* Access to payment processor

## Steps

### 1\. Generate Payout Report

```
1. Go to Admin > Payouts
2. Select date range (last 7 days)
3. Export CSV
```

### 2\. Verify Eligibility

Check each trader meets:

- [ ] Minimum balance threshold
- [ ] KYC completed
- [ ] No pending disputes

### 3\. Process Payments

1. Upload CSV to payment processor
2. Review totals match
3. Approve batch

### 4\. Update Records

* Mark payouts as processed in admin
* Send confirmation emails

## Troubleshooting

| Issue | Solution |
| -- | -- |
| Payment failed | Check bank details |
| Amount mismatch | Recalculate from trades |
