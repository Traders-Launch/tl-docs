# Customer Support Playbook

## Common Issues & Responses

### Account Access

**Issue**: Can't log in
**Response**: Reset password link, check email spam

### Billing

**Issue**: Charged incorrectly
**Response**: Verify in Stripe, process refund if needed

### Technical

**Issue**: App not loading
**Response**: Clear cache, try different browser, check status page

## Escalation Path

1. Support (Discord/email)
2. Connor (technical issues)
3. Zaeem (engineering bugs)

## Response Time Targets

| Priority | Target |
| -- | -- |
| Urgent | 2 hours |
| Normal | 24 hours |
| Low | 48 hours |
