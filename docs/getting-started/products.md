# Allowed Trading Products

Traders Launch supports futures trading across major CME Group exchanges.

## Supported Exchanges

- **CME** - Chicago Mercantile Exchange
- **CBOT** - Chicago Board of Trade
- **NYMEX** - New York Mercantile Exchange
- **COMEX** - Commodity Exchange

## Position Limits

Position limits vary by account size. See [Rules](/getting-started/rules) for details.

## Restricted Products

Some products have additional restrictions:
- **Cryptocurrency futures** - Not currently supported
- **Options on futures** - Not supported
- **Spreads** - Calendar spreads only
